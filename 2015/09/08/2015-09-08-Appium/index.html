Appium环境安装简化版本

  • 自动化
Learn

准备前期

现在说下现在的Appium的情况，现在在github上的例子有点老旧了，有点不适用了，但还是可以用来跑跑demo的，地址放出来

https://github.com/appium/sample-code/tree/master/sample-code

下面就是简单的说下怎么环境怎么安装了，网上有很多很详细的说明，这个简单的主要做为自己的笔记使用

一、安装基本的Android环境：

JDK、ADT(eclipse/Adroid的开发)、SDK(安卓的系统)

二、安装Node.js:

下载地址：https://nodejs.org
检查是否安装成功的方法：进入CMD，输入node -v

三、安装Appium的服务器端（Appium Desktop Apps）：

不分语言版本，只分平台
两种方式 ：
1、 在线安装：要求必要先安装好Node，直接CMD下输入:npm install –g appium
2、 下载安装：
官方网址：

http://appium.io/downloads.html

下载地址：

https://bitbucket.org/appium/appium.app/downloads/

安装完后，手动去设置

d:\XX\Appium\node_modules.bin\安装目录为环境变量

检查安装是否完整方法：CMD下输入appium-doctor

如图显示为全成功，就表示服务端环境没有问题了

四、下载客户端（Appium Client Libraries）：

下载地址：http://appium.io/downloads.html
看自己用的语言版本下载就行了，用来导入的，不用安装

五、创建Java工程

（1）导入Selenium类库
http://selenium-release.storage.googleapis.com/2.44/selenium-java-2.44.0.zip
http://selenium-release.storage.googleapis.com/2.44/selenium-server-standalone-2.44.0.jar
（确认2014.11.14有效，如无效请访问http://docs.seleniumhq.org/download/ 进行下载 ）
（2）导入Appium类库
四下来的导入到java工程中

六、运行例子

先打开任何一个cmd输入appium，启动服务端

去下载例子吧。
https://github.com/appium

直接运行例子(有各种语言版本的，还有对应的App都在上面)
