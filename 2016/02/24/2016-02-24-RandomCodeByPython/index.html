python生成随机码

  • python
Learn

代码

直接把这个生成方法的代码放出来吧，原博客的地址：好人一生平安

分析

博客中也推荐了别人写的一篇别人写的生成随机码的方法，其实主要有两个问题

  1. 生成的随机码不重复
  2. 别人使用随机码时能否方便查询是否正确

这里使用的方式是主键+随机码的方法
代码中使用“L”做为标识符，区别主键和随机码，主键唯一，就可以很快查询到主键生成的随机码是否正确
上面我就解释下两个方法

prefix = hex(int(id))[2:]+ ‘L’

hex是把对应的ID生成十六进制的数，但会在前面加上0X，表示这个数是16进制的，这里只是去掉这个16而已

return prefix + ‘’.join([random.choice(chars) for i in range(length)])

这个是简写的方式

剩下的不用解释了，这么写也就知道了

return str(int(code.upper(), 16))

这个也是刚刚学的，直接把16进制的转成10进制，就是要这么写，简单实用
